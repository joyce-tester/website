Python Course Plan

Week 1: Introduction to Python

A beginner-friendly introduction to Python basics and setup.

  • Setting up Python and IDE
  • Writing your first Python program
  • Python syntax, variables, and comments

Week 2: Python Basics

Learn the essentials of Python programming, including input/output and control flow.

  • Basic arithmetic and comparison operations
  • Control flow: if, elif, else
  • String operations and slicing

Week 3: Loops and Functions

Master loops and learn to write reusable functions in Python.

  • Loops: for and while
  • Defining and calling functions
  • Scope of variables
